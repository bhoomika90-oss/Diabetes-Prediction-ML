# Diabetes Prediction using Machine Learning

## Overview
This project predicts whether a person is diabetic or not using Machine Learning techniques.

## Problem Statement
Early detection of diabetes helps in preventing severe health complications.

## Dataset
- PIMA Indians Diabetes Dataset
- Features include glucose level, blood pressure, BMI, age, etc.

## Machine Learning Model
- Logistic Regression (or mention your exact model)

## Technologies Used
- Python
- Flask
- Pandas, NumPy
- Scikit-learn
- HTML, CSS

## Working
1. Data preprocessing and cleaning
2. Model training
3. Model saving using Pickle
4. Web interface for prediction

## Result
The system predicts whether the patient is diabetic or non-diabetic.

## Conclusion
The ML-based system provides an efficient way to predict diabetes with good accuracy.
